1.2写入口函数 和 不写入口函数 的区别？

  1. 如果写了入口函数 那么不管元素对象是在后边或者上边 都能获取到
  2. 如果不写入口函数 那么久会按照由上往下的顺序执行代码 如果元素对象写在了js的后边那么还是获取不到

1.3 jQuery 对象和 DOM 对象

使用 jQuery 方法和原生JS获取的元素是不一样的，总结如下 :

  1. 用原生 JS 获取来的对象就是 DOM 对象
  2. jQuery 方法获取的元素就是 jQuery 对象。
  3. jQuery 对象本质是： 利用$对DOM 对象包装后产生的对象（jquery对象就是一个伪数组，存储的是一个一个的DOM对象 ）。
  4. 只有 jQuery 对象才能使用 jQuery 方法，DOM 对象则使用原生的 JavaScirpt 方法。

9.0 jQuery 效果

  • 显示隐藏：show() / hide() / toggle() ;
  • 滑入滑出：slideDown() / slideUp() / slideToggle() ;
  • 淡入淡出：fadeIn() / fadeOut() / fadeToggle() / fadeTo() ;
  • 自定义动画：animate() ;

注意：

动画或者效果一旦触发就会执行，如果多次触发，就造成多个动画或者效果排队执行。

jQuery为我们提供另一个方法，可以停止动画排队：stop() ;

9.1显示隐藏(对角线)

显示隐藏动画，常见有三个方法：show() / hide() / toggle() ;可带参数可不带参数详情查api

9.2滑入滑出

滑入滑出动画，常见有三个方法：slideDown() / slideUp() / slideToggle() ; 详情查api

9.3 淡入淡出

淡入淡出动画，常见有四个方法：fadeIn() / fadeOut() / fadeToggle() / fadeTo() ;

fadeTo 改变透明度 要求：必须要写两个参数 第一个是时间 第二个是透明度

9.4 自定义动画

  • 停止动画排队:stop() 方法用于停止动画或效果

  • stop() 写到动画或者效果的前面

  • 每次使用动画之前，先调用 stop() ,在调用动画

    参数有四个

    第一个参数是： 想要利用动画来修改的样式属性 要以对象的形式传递过来

    第二个参数是： 在执行这个动画开始到结束的一个时间

    第三个参数是： 切换状态 – 运动速度

    第四个参数是： 回调函数 — 等这个动画执行完毕以后 就执行这个函数(回头执行函数)

9.5 事件切换

  • over:鼠标移到元素上要触发的函数（相当于mouseenter）
  • out:鼠标移出元素要触发的函数（相当于mouseleave）
  • 如果只写一个函数，则鼠标经过和离开都会触发它
